# Elagi API Testing

## About The Project

This project is a Postman collection created to test the APIs of Elagi application.

It covers the main features of the system like authentication, products, cart, and checkout.
The goal is to make sure all APIs are working correctly.

---

## Features Tested

### Authentication

* Register
* Verify Email (OTP)
* Login

### Password

* Forgot Password
* Reset Password

### User Profile

* Get user data
* Update user data

### Products

* Get products by category
* Pagination
* Sorting by price

### Wishlist

* Add to wishlist
* Get wishlist
* Remove from wishlist

### Cart

* Add items to cart
* Get cart items
* Update quantity
* Delete item

### Checkout

* Place order

---

## Tools Used

* Postman
* REST API
* JavaScript (for test scripts)

---

## How to Run

1. Open Postman
2. Import the collection file
3. Import the environment file
4. Select the environment
5. Run the requests

---

## Notes

* Replace sensitive data like email and token before running
* Make sure the base URL is correct
